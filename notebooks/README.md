# Introduction

NumPy est une bibliothèque Python puissante pour le calcul scientifique et la manipulation de tableaux multidimensionnels. Elle permet de réaliser des opérations mathématiques rapidement et efficacement grâce à des tableaux appelés **ndarrays**.

Pour visualiser vos données, nous utilisons également **Matplotlib.pyplot (Pyplot)**, qui permet de tracer facilement des graphiques linéaires, des histogrammes, des diagrammes en barres, des nuages de points, et bien d’autres types de visualisations.

Les exercices proposés permettent de se familiariser avec :  

* La création de tableaux NumPy  
* Les fonctions mathématiques sur les tableaux  
* Les manipulations de tableaux (tri, filtrage, opérations logiques)  
* La visualisation de données avec Pyplot (graphes, histogrammes, barres, nuages de points, courbes sinusoïdales, polynômes, boxplots, etc.)  

Pour faciliter l'apprentissage et la vérification de vos solutions, le projet contient trois supports complémentaires :  

* Le **fichier Jupyter Notebook `exo_numpy.ipynb`**, avec les exercices NumPy à réaliser et les résultats à tester directement.  
* Le **fichier Jupyter Notebook `exo_pyplot.ipynb`**, avec les exercices Pyplot à réaliser et les graphiques à générer.  
* Le **fichier de snippets VSCode `.vscode/numpy_snippets.json`**, contenant les solutions NumPy sous forme de fonctions prêtes à insérer dans vos scripts.  
* Le **fichier de snippets VSCode `.vscode/pyplot_snippets.json`**, contenant des fonctions Pyplot prêtes à tracer vos graphiques avec des exemples commentés.
